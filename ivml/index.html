IVML

Angular.js directives for data visualization with d3.js
IVML is a JavaScript library that leverages popular JavaScript technologies Angular, D3 and JQuery. It uses d3js to provide embeddable AngularJS directives for data visualization, greatly reducing the effort required to make interactive dashboards.
